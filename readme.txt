This is a plugin that helps create previews of a url based on the OGP of the page, similar to a url preview in a Facebook post.

== Description ==

The plugin works with a URL of plain text, will fetch title and summary and thumbnail from URL.

= Contribute! =
You can fork the plugin from [GitHub](https://github.com/kuck1u/va-wsd-the-phantom-thief)

== Installation ==

To install VA WSD the phantom thief:

1. Upload the "va-wsd-the-phantom-thief" directory and all its contents to your `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

== Screenshots ==

1. Paste a URL in visual editor.
2. Embed link with title and summary and thumbnail.
